3 Hướng Build Pipeline TikTok

7+ video/tuần, fully automated, VI+EN, robot mascot Milo

A

All-Cloud APIs (Plug & Play)

Stack: Claude API → ElevenLabs → Runway/Kling → FFmpeg → TikTok API

Orchestration: n8n self-hosted (free)

Pros

  • Setup nhanh, ít code
  • Quality cao nhất
  • Scale dễ

Cons

  • Chi phí cao nhất ($100-300/tháng ở 7 vid/tuần)
  • Phụ thuộc 3rd party uptime
  • Ít điểm "AI engineering" trên CV

Chi phí ước tính ~$150-300/tháng

B

Local-First (Open Source Heavy)

Stack: Ollama (local LLM) → ComfyUI + SD LoRA → AnimateDiff → Edge-TTS → FFmpeg

Orchestration: Python scripts + cron

Pros

  • Chi phí gần $0 (chỉ điện)
  • Full control
  • Privacy tốt

Cons

  • Cần GPU mạnh (≥8GB VRAM)
  • Setup phức tạp
  • Video quality thấp hơn A
  • Tiếng Việt TTS kém hơn

Chi phí ước tính ~$0-10/tháng + hardware

C

⭐ Hybrid Orchestrated Pipeline (Khuyến nghị)

Stack:

  • 📝 Script: Claude API (prompt caching → rẻ nhất)
  • 🎨 Image: FLUX via Replicate (pay-per-use, no subscription) + LoRA Milo
  • 🎬 Video: Kling AI (tốt nhất giá/chất lượng 2025) hoặc Hailuo
  • 🔊 TTS: Edge-TTS (free, VI+EN tốt) + ElevenLabs fallback
  • ✂️ Assembly: FFmpeg + Remotion
  • 📅 Schedule: n8n self-hosted + TikTok API

Pros

  • Chi phí tối ưu: ~$30-80/tháng ở 7 vid/tuần
  • Quality cao (Kling = top-tier video AI)
  • Edge-TTS free VI+EN chất lượng tốt
  • Claude caching giảm 80% API cost
  • n8n visual workflow = CV đẹp
  • Dễ swap từng component

Cons

  • Setup 1-2 tuần đầu
  • Cần maintain Milo LoRA

Chi phí ước tính ~$30-80/tháng
