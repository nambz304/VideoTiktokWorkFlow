Thách thức chính: giữ Milo nhìn như nhau qua các ảnh AI gen khác nhau.

🎯
FLUX LoRA — Train riêng cho Milo
Dùng 3-10 ảnh ref Milo để train LoRA (~15-30 min). Sau đó mọi lần gen đều dùng LoRA này → Milo luôn nhất quán 95%+. Một lần train, dùng mãi.

Workflow: Train LoRA trên Replicate (flux-lora-training) → deploy → mỗi cảnh gọi FLUX + LoRA + script prompt
Train: ~$2-5 một lần Gen: $0.05/img Consistency: 95%+ 10-15s/img
🖼
FLUX + IP-Adapter (flux-pulid)
Truyền ảnh ref Milo vào IP-Adapter mỗi lần gen. Không cần train — chỉ cần ảnh ref tốt. Nhưng consistency thấp hơn LoRA (~80%), đặc biệt với nhân vật hoạt hình/robot.
$0.05-0.08/img Consistency: ~80% Drift với robot/cartoon ~15s/img
⚙️
ComfyUI tự host + ControlNet + LoRA
Full control: LoRA + ControlNet (pose) + IP-Adapter kết hợp. Milo gần như không drift. Nhưng cần server riêng và setup phức tạp.
Server: $20-50/tháng Consistency: 98% Setup phức tạp 5-10s/img (GPU server)
Recommend cho Milo
Approach B + FLUX LoRA:
1. Lấy 5-10 ảnh Milo reference (đã có Milo_reference.jpeg + 10 pose PNG)
2. Train LoRA 1 lần trên Replicate → $2-5, ~20 phút
3. Mỗi scene: FLUX + LoRA + prompt từ kịch bản → gen 1 ảnh tích hợp BG + Milo đúng hành động
4. Không cần rembg, không cần composite PIL

Chi phí thực tế: $0.05 × 5 cảnh = $0.25/video. Với 100 video/tháng = $25/tháng.
